Help

Map movement:

  • Hold down the middle mouse button and drag the mouse.
  • Hold down the right mouse button and drag the mouse.

Zoom in and out:

  • Scroll up with the mouse cursor on the map to zoom in.
  • Scroll down with the mouse cursor on the map to zoom out.

Play time:

  • Scroll up on the time with the mouse cursor to increase the time.
  • Scroll down on the time with the mouse cursor to decrease the time.

  • Left-click on the + to increase the time.

  • Left-click on the - to decrease the time.

Toolbox:

  • Resets the zoom and position of the map.

  • Activates/Deactivates the drawing mode.

  • On the map, hold down the left mouse button to draw.

  • Allows you to change the color of the line.

  • Allows you to change the thickness of the line.

  • Allows you to undo the last drawing actions.

  • Allows you to redo the undone drawing actions.

  • Deletes all drawings.

Element categories:

  • Left-click on the arrows ▲ ► ▼ ◄ to open/close the categories.
  • Use the horizontal scroll to see hidden elements.
  • Hovering over an element for more than one second displays its name on the screen.

Playable Pokémon Category:

  • Can be dropped on the map.
  • Hold down the left mouse button on an element and drag the mouse to slide it.
  • Release the left mouse button on an element to drop it.
  • An element released outside the map returns to its category.
  • Right-click on a dropped element to store it in its category.

Neutral Pokémon Category:

  • Can be dropped on the map.
  • Hold down the left mouse button on an element and drag the mouse to slide it.
  • Release the left mouse button on an element to drop it.
  • An element released outside the map is deleted.
  • Right-click on a dropped element to delete it.

Battle Items and Held Items Category:

  • Can be dropped on the map.
  • Can be dropped on a playable Pokémon.
  • Only one Battle Item can be attached to a playable Pokémon.
  • Three Held Items can be attached to a playable Pokémon.
  • Hold down left-click on an item and move the mouse to drag it.
  • Release left-click on an item to drop it.
  • An item dropped outside of the map is removed.
  • Right-click on a dropped item on the map to remove it.
  • Left/Right/Middle-click on a dropped item on a playable Pokémon to remove it.

Miscellaneous Category:

  • Purple team's base destroyed.

  • Orange team's base destroyed.

  • Rayquaza's whirlwind.

Aide

Déplacement de la carte :

  • Maintenir enfoncé clic molette et déplacer la souris.
  • Maintenir enfoncé Clic droit et déplacer la souris.

Zoom et dézoom :

  • Défilement vertical haut avec le curseur de la souris sur la carte pour zoomer.
  • Défilement vertical bas avec le curseur de la souris sur la carte pour dézoomer.

Temps de jeu :

  • Défilement vertical haut avec le curseur de la souris sur le temps pour augmenter le temps.
  • Défilement vertical bas avec le curseur de la souris sur le temps pour diminuer le temps.

  • Clic gauche sur le + pour augmenter le temps.

  • Clic gauche sur le - pour diminuer le temps.

Boîte à outils :

  • Réinitialise le zoom et la position de la carte.

  • Active/Désactive le mode dessin.

  • Sur la carte, maintenir enfoncé clic gauche pour dessiner.

  • Permet de changer la couleur du trait.

  • Permet de changer l'épaisseur du trait.

  • Permet d'annuler les dernières actions du dessin.

  • Permet de rétablir les actions annulées du dessin.

  • Supprime tous les dessins.

Catégories d'éléments :

  • Clic gauche sur les flèches ▲ ► ▼ ◄ pour ouvrir/fermer les catégories.
  • Utiliser le défilement horizontal pour voir les éléments cachés.
  • Survoler un élément pendant plus d'une seconde affiche son nom à l'écran.

Catégorie Pokémon jouables :

  • Peut être déposé sur la carte.
  • Maintenir enfoncé clic gauche sur un élément et déplacer la souris pour le glisser.
  • Relâcher clic gauche sur un élément pour le déposer.
  • Un élément relâché en dehors de la carte revient dans sa catégorie.
  • Clic droit sur un élément déposé pour le ranger dans sa catégorie.

Catégorie Pokémon Neutres :

  • Peut être déposé sur la carte.
  • Maintenir enfoncé clic gauche sur un élément et déplacer la souris pour le glisser.
  • Relâcher clic gauche sur un élément pour le déposer.
  • Un élément relâché en dehors de la carte est supprimé.
  • Clic droit sur un élément déposé pour le supprimer.

Catégorie Objets de combat et Objets tenus :

  • Peut être déposé sur la carte.
  • Peut être déposé sur un Pokémon jouable.
  • Un seul Objet de combat peut être attaché à un Pokémon jouable.
  • Trois Objets tenus peuvent être attachés à un Pokémon jouable.
  • Maintenir enfoncé clic gauche sur un élément et déplacer la souris pour le glisser.
  • Relâcher clic gauche sur un élément pour le déposer.
  • Un élément relâché en dehors de la carte est supprimé.
  • Clic droit sur un élément déposé sur la carte pour le supprimer.
  • Clic gauche/droit/molette sur un élément déposé sur un Pokémon jouable pour le supprimer.
